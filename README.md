#URL Shortener Web Application

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).

A responsive URL Shortener built with React.js and Firebase Realtime Database. The application allows users to convert long URLs into short, shareable links with optional custom aliases while validating inputs and preventing duplicate aliases.

🚀 Features
 Shorten long URLs into compact links
 Create custom aliases (optional)
 URL format validation
 Duplicate alias detection
 Store shortened URLs in Firebase Realtime Database
 Copy generated URL to clipboard
 Responsive UI using Bootstrap
 Loading indicator during URL generation
